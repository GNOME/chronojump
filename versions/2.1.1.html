This is a recommended upgrade for all our force sensor and race analyzer users. Changes since 2.1.0-36 (2021 Feb 8)

General Chronojump changes

  • Any session can be edited from load session view.
  • Chronojump can be shown now in an low height device (480 px).
  • Chronojump can be run now on machines with less CPU.
  • Persons on top window management allows to see persons with or without images, and delete person has moved. So now is more usable and smaller.
  • Preferences window has a better tabs viewing selection, and is now resizable.
  • Fixed size of dialog on confirmation of import session.

Force sensor

  • Analyze with 3 modes: Individual current set, individual current session, groupal current sesssion.
  • Individual current session and groupal current session exports all data including images (of set and RFD model) in an automated way. Note is important to have a correct exercise configuration on detect repetitions tab in order to export automatically data. Exported CSV shows person name, date, exercise, laterality, set, repetition, comments and obviously measured variables. Graphs size is also configurable.
  • Implemented on analysis maximum average force in X seconds, being X a value configurable on preferences. It is shown the value on the program, a green segment on the graph and this new variable is also exported.
  • On exercise dialog, better display of detect repetitions by preferences or custom.
  • On analyze now there is a chain button to link A and B sliders (AB slider has been removed).
  • Analyze sliders +1s (or user configured value on preferences) now can adjust better wich sample to use, just immediately after or before selected time, whatever is closer.
  • Calibration minimum value from 1 to 0.01
  • On preferences added two sub tabs for force sensor: capture/analyze
  • Fixed a bug that could lead under certain conditions to force data loss of a person if other person of same session is deleted.
  • Fixed wrong Minimum concentric/eccentric values for raw measure on "elastic" (program allowed to configure a weird option that now is not possible).

Race analyzer

  • Analyze with 3 modes: Individual current set, individual current session, groupal current sesssion.
  • Individual current session and groupal current session exports all data including images in an automated way. Exported CSV shows person name, date, exercise, comments and obviously measured variables. Graphs size is also configurable.
  • Fixed a bug that could lead under certain conditions to race analyzer data loss of a person if other person of same session is deleted.
  • Updated meters/pulse from 0.003003 to 0.0030321 to have a more accurated capture.
  • PPS are always sent to the device at the beginning of the capture.
  • Now in preferences PPS configuration it is shown the max speed for measuring at that time accuracy.

Jumps, Races

  • Added an option to show time on any run simple graph.
  • Hidden run/run interval execution option with reaction time measurement.

Chronojump-networks

  • Security improvements on all the system.

Translations

  • Updated translations: Brazilian portuguese, Catalan, Spanish, Swedish, Ukrainian.
