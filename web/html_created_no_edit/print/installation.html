If you are an advanced user: from an operative system GNU/Linux, install Mono, download the Chronojump sources on the chronojump SVN on GNOME, and execute mono chronojump.exe or compile with make clean, make.

Help can be found reading the manual (spanish), subscribing to the mailing lists. We recommend you to read the FAQ.

If you wish to know the advance of the project, please check on changelog file.
