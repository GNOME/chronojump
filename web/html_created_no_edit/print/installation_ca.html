Ja podeu provar ChronoJump 0.71 (27-Maig-2008) a qualsevol PC. Aquesta versió és totalment usable. Consulteu el llistat de canvis respecte l'anterior versió.

Si us considereu un usuari avançat, podeu provar el programa des d'un sistema operatiu GNU/Linux, instal·lant Mono, baixant-vos les fonts del SVN de Chronojump a GNOME així: svn checkout http://svn.gnome.org/svn/chronojump/trunk chronojump.

Podeu obtenir ajuda llegint el manual (castellà), apuntant-vos a les llistes de correu. Es recomana que llegiu les preguntes freqüents.

Si desitgeu conèixer com avança el projecte, podeu consultar l'arxiu changelog.
