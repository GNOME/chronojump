Installation

Printable version
Now you can try ChronoJump 0.8 (December-2-2008) from any PC. This version is fully usable. Read changes list here.

Operating systems
Installation
GNU/Linux Chronojump 0.8 Linux
Windows Chronojump 0.8 Windows
Attention, if after installation, Chronojump doesn't start at first execution, try it two or three times more and it will work. This bug only exists in some computers. We hope it will be fixed soon in a new version.
Maemo N770Read this e-mail: chronojump_mini for Maemo!! Chronojump_mini funcionando en Maemo!!


If you are an advanced user: from an operative system GNU/Linux, install Mono, download the Chronojump sources on the chronojump SVN on GNOME doing: svn checkout http://svn.gnome.org/svn/chronojump/trunk chronojump..

Help can be found reading the manual (spanish), subscribing to the mailing lists. We recommend you to read the FAQ.

If you have any problem with the software, check Problems page.

If you wish to know the advance of the project, please check on changelog file.

License

ChronoJump is free software  distributed under the GPL license. The program and chronometer source code is available, it is also possible to modify and redistribute.

The chronometer and the contacts platform come along with construction guidelines.

It is not possible to verify the validity and fiability of a digital measuring instrument if the source code is not available. Therefore Chronojump is appopriate to investigations.
