Passos per a isntal·lar chronojump_django:

1 - Instal·lació de paquets python 

#cd /srv/chronojump-server/requirements/django
#pip install wheel
#pip install -r local.txt


2 - Crear una base de dades a mysql o mariadb 

NO pots fer servir l'antiga, si això és un requisit, llavors no podeu fer servir Django de moment. En cas contrari crea una nova base de dades. Una vegada fet, necessitem aquesta variable d'entorn:

DATABASE_URL=mysql://user:password@host:port/db_name

Exemple:
export DATABASE_URL="mysql://chronojump:chronojump@127.0.0.1:3306/chronojump-django"

crear base de dades a mysql:
#mysql
    > create database chronojumpserver_django

A la carpeta de chronojumpserver_django executar:
#export DATABASE_URL="mysql://root:passwordbarca@localhost:3306/chronojump_django" (Utilitzar el 127.0.0.1 o 0.0.0.0 com a host resultava en error en el migrate, utilitzant localhost ho soluciona).
#python manage.py migrate


3 - Iniciar Django

A la carpeta chronojumpserver_django executa el següent:

Crear o actualitzar taules:
#python manage.py migrate

Compilar les traduccions dels idiomes:
#python manage.py compilemessages

Crear l'usuari admin de chronojump:
#python manage.py createsuperuser

Iniciar django:
#python manage.py runserver 0.0.0.0:8000


Per accedir a django escriure al navegador: http://localhost:8000

Entrarem amb la conta del superusuari creada:

user: admin
passwd: adminfucku


4 - Consola d'administració on pots crear tot
http://localhost:8000/admin (fes servir el usuari d'abans creat)
