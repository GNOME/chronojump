1. Install MSYS2

https://github.com/msys2/msys2-installer/releases/download/2023-10-26/msys2-x86_64-20231026.exe

2. Open "MSYS2 UCRT64" shell

3. Execute command: pacman -Syu

4. Install autotools: pacman -S autoconf automake libtool base-devel

5. Install git and pkg-config: pacman -S git pkg-config

6. Install intltoolize: pacman -S intltool

7. Install GCC: pacman -S mingw-w64-ucrt-x86_64-gcc

8. Install Python and glib: pacman -S python3 mingw-w64-ucrt-x86_64-glib2 mingw-w64-ucrt-x86_64-gettext

9. Install gtk-sharp

brew install mono
git clone -b main https://github.com/mono/gtk-sharp.git
//Open gtk-sharp/gtk/gui-thread-check/profiler/gui-thread-check.c and comment LINE 98-100.
cd gtk-sharp
sh autogen.sh --prefix=`pkg-config --variable=prefix mono`
make
make install
