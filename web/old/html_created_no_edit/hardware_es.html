2.- Construcción de la plataforma de contactos
El instrumento que permite saber si el sujeto está en el aire o en el suelo puede ser una plataforma de contactos o un sistema de células fotoeléctricas.
Construcción de plataformas de contactos
Modular de circuito impreso De varillas
Informaciones antiguas:
  • Antes del proyecto ChronoJump, se crearon los proyectos Salta y Gsalta (hoy día desaparecidos), en éstos no existía cronómetro, y la conexión plataforma-PC se producía por el puerto paralelo. Aunque este método no se adapta a ChronoJump se incluye para quienes estén interesados: información de conexión por puerto paralelo.
  • Alternativamente se ha descrito otra posibilidad de construcción de plataforma de contactos que no ha sido probada o adaptada a ChronoJump: "Desarrollo de Prototipo de Plataforma y Programa de Computación, para Evaluación de Salto Vertical adaptado de Tapete de Videojuego"

Chronojump gana premio de software libre 2007 en la categoría de educación.

Licencia

ChronoJump es software libre distribuido bajo la Licencia GPL. El código fuente del programa y del cronómetro puede ser consultado, así como modificado y redistribuido.

Se facilitan instrucciones de construcción de la plataforma de contactos y el cronómetro.

No es posible verificar la validez y fiabilidad de un instrumento informático de medida si no se facilita el código fuente. Por esta razón ChronoJump es óptimo para la investigación.
